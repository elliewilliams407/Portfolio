Creating Your Own Pixel

Reimagining the pixel as a building block for digital art

About the Project

Pixels are the fundamental building blocks of digital images and displays. Conventionally, they are uniform squares arranged in a static Cartesian grid. This project challenges that assumption by reconstructing images using custom picture elements.

The code takes in a folder of smaller images, which are then used as the "pixels" to recreate a chosen main image. The result is a mosaic-like artwork where each pixel tells its own story.

Technologies Used

  • Processing – for creative coding and image rendering
  • Python – for image processing and automation
