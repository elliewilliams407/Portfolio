Concentration Game

About the Project

This game was created to practice mutable world states, ArrayLists, mutable data structures and loops.

Steps Taken To Complete Project

    1. Conceptualized the Game

    2. Setup project in Eclipse IDE and structured the project with necessary classes and methods

    3. Implemented classes, developed game logic, and implemented key features

    4. Handled user interaction

    5. Designed the user interface and added difficulty modes

How to Run

To play the game:

  1. Clone the repository (or download it as a ZIP).
  2. Open the project in an IDE like Eclipse.
  3. Set up dependencies (like the Big Bang library, if required).
  4. Compile and run the game.
