Concentration Game

A memory card-matching game built with Java & Big Bang

About the Project

This game was developed to practice mutable world states, ArrayLists, loops, and mutable data structures. The player flips two cards per turn, with the goal of finding all matching pairs. The game includes difficulty modes and a user-friendly interface.

Steps Taken

  • Conceptualized the overall game mechanics and flow.
  • Structured the project in Eclipse IDE with appropriate classes and methods.
  • Implemented game logic, core classes, and key features.
  • Added player interaction handling and difficulty options.
  • Designed a clean, interactive user interface for gameplay.

How to Run

  1. Clone the repository (or download as a ZIP).
  2. Open the project in Eclipse or another Java IDE.
  3. Set up dependencies (such as the Big Bang library).
  4. Compile and run the game.
