NomNetwork Project

About the Project

This project was playing around with applications in order to put an entire application together using realistic data generation, planning and developing our REST API, and implementing a set of UI screens in Streamlit. The project allows a persion to interactively explore different user stories for the suggested app and how these personas differ in what they require from the app's function.

Steps Taken To Complete Project

    1. Designed a relational database model

    2. Wrote simple and complex CRUD SQL against that database model

    3. Accessed and queried a database from programming languages (Python and Java)

    4. Performed a conceptual modeling and logical modeling for our app

    5. Built a simple CRUD application using a RDBMS in the back-end
