Concentration Game

Concentration is a card game in which all of the cards are laid face down. At each turn, two cards are flipped over. The object of the game is to find pairs of matching cards. In this version, there is only one player. This was created using java and Big Bang.

Drawing Machine

This project's purpose was to form a true human-machine collaboration. The user has somewhat artistic control as they draw but the computer randomly errases work forcing an abstract outcome. Th user is free drawing with colors, shapes, and thicknesses while the computer alters the drawing with its own random shapes, directions, and durations.
