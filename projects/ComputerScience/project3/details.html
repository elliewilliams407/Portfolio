Drawing Machine

About the Project

This is a take on "Make your own paintbrush." The exercise personalizes and defamiliarizes the act of mark-making, and invites a deeper consideration of how tools and technologies shape artistic expression. This Drawing Machine assignment was meant to capture this spirt in the domain of software, opening up questions of constraints, autonomy, and augmentation in human-machine collaboration.

Technologies Used:

    - Processing

    - Python
