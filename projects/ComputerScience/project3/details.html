Drawing Machine

Exploring human-machine collaboration through generative art

About the Project

This project reimagines the classic "Make your own paintbrush" exercise in a digital space. It personalizes and defamiliarizes the act of mark-making, encouraging reflection on how tools and technologies shape artistic expression. The Drawing Machine highlights the tension between constraints, autonomy, and augmentation in human-machine collaboration, allowing users to create art while the computer dynamically alters it.

Technologies Used

  • Processing – for interactive generative visuals
  • Python – for scripting behaviors and logic
